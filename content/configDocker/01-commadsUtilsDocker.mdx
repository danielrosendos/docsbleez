## Tabela De Comandos


| Comando                                     | O que faz                               | 
| ------------------------------------------- |:---------------------------------------:|
| docker ps                                   | lista os containers ativos              |
| docker ps -a                                | lista todos os containers instalados    |
| docker exec -it nomecontainer /bin/bash     | abre o terminal daquele containe        |

Esta sessão irá mostrar os passos da instalação o docker compose em ambientes
linux. A documentação oficial do docker compose está neste link [Docker Composer](https://docs.docker.com/compose/install/).

### Instalando Compose em Sistemas linux

1.  Execute este comando para baixar a versão estável atual do Docker Compose:

    ```bash
    sudo curl -L "https://github.com/docker/compose/releases/download/1.25.4/docker-compose-$(uname -s)-$(uname -m)" -o /usr/local/bin/docker-compose
    ```

2.  Aplique as permissões necessárias no binario

    ```bash
    sudo chmod +x /usr/local/bin/docker-compose
    ```

3.  Verificando se o docker compose foi instalado corretamente

    ```bash
    $ docker-compose --version
    docker-compose version 1.25.4, build 1110ad01
    ```
